# CivicLens Frontend

React + Vite app for the CivicLens NYC 311 analytics dashboard.

See the [project README](../README.md) for an overview, features, and setup.

## Quick start

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Scripts

- `npm run dev` — development server
- `npm run build` — production build
- `npm run preview` — preview production build
- `npm run lint` — lint source files
